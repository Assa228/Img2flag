# Img2flag                                                         


# Install :

1. ```sudo apt install tesseract-ocr```

2. ```sudo apt install libtesseract-dev```

3. ```git clone https://github.com/Assa228/img2flag.git```

4. ```cd img2flag```

5. ```pip install -r requirements.txt```


# Usage :
* ```python img2flag.py -f flag.png```
